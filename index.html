One-time setup (10 minutes)
1

Make Raw-Data a table. In Excel, click any cell in Raw-Data → Insert → Table (has headers) → Table Design → rename it RawData.

2

Register a free Azure app. Go to portal.azure.com → Microsoft Entra ID → App registrations → New. Supported account types: Personal Microsoft accounts. Platform: Single-page application. Redirect URI: the exact URL where this page is hosted.

3

Add permission. API permissions → Microsoft Graph → Delegated → Files.ReadWrite.

4

Host this file. Sign-in can't run from a file:// path. Put this HTML on GitHub Pages (free, HTTPS) and use that URL as the redirect URI in step 2.

5

Paste the client ID above, set the workbook path (e.g. /Documents/BUDGET.xlsx), sign in once per phone. Both of you use the same client ID; you each sign in with the Microsoft account that has access to the file.

How this app decides categories

It ships with a memory of merchants learned from your history (Dec 2023 → Feb 2026). Typing a known name auto-fills category and sub-category. Every entry you save teaches it new names. Category strings are written back exactly as your sheet expects (including the trailing space in Essential␠ and Investment␠) so your DashBoard formulas keep working.
